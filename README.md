# MarkText v0.17.1简体中文汉化版

开源 Markdown 编辑器 · 完整简体中文本地化

自 AI Agent 流行以来，Markdown 逐渐成为大模型知识库与文档沉淀中非常理想的基础格式。我也一直在寻找一款好用、开源、免费的 Markdown 文档编辑器，直到遇见了 MarkText。

MarkText 是一款非常优秀的软件，简洁、专注、体验出色，几乎完美契合了我对 Markdown 编辑器的所有需求。在此，向开源社区以及 MarkText 的所有贡献者表示感谢。

作为一名简体中文用户，我深知语言门槛常常横亘在优秀开源软件与国内用户之间。因此，我制作了这个简体中文汉化版，希望能让更多中文用户更轻松地使用 MarkText。

目前，相关功能均已完成测试。就个人体验而言，本次汉化覆盖已经比较完整。

## 安装方法

### macOS

打开 `.dmg` 文件，将 MarkText 拖入「应用程序」文件夹。

首次打开如果提示「无法验证开发者」，需要在「系统设置 → 隐私与安全性」中点击「仍要打开」。

---

## 汉化说明

本版本基于 MarkText v0.17.1 源码进行中文本地化，所有界面文字已翻译为简体中文：

- 主菜单栏、右键菜单
- 偏好设置（全部 7 个页面）
- 快捷键描述
- 选中文字的格式工具栏提示
- 退出保存对话框

**注意**：偏好设置中的「界面语言」下拉框为只读选项，本版本固定为简体中文，不支持切换语言。

## 从源码构建

```bash

cd marktext-zh
npm install --legacy-peer-deps
npm run release:mac      # macOS
```

构建产物在 `build/` 目录下。
